## 🎨 主题说明

本博客使用了 [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) 主题，这是一个功能丰富、响应式的 Jekyll 主题，具有以下特点：

- 📱 完全响应式设计
- 🌙 深色/浅色主题切换
- 🔍 全文搜索功能
- 📊 Google Analytics 集成
- 💬 多种评论系统支持
- 🏷️ 标签和分类系统
